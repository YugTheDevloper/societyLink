# SocietyLink: Smart Society Management System 🏢

A robust backend REST API built to digitize and streamline operations for residential societies. This system handles secure resident authentication, profile management, and an automated complaint ticketing system.

## 🚀 Tech Stack
* **Language:** Java 
* **Framework:** Spring Boot
* **Database:** MySQL / Spring Data JPA
* **Build Tool:** Maven

## ⚙️ Key Features
* **Resident Management:** Complete CRUD operations for onboarding and managing society residents.
* **Complaint Resolution:** A ticketing system allowing residents to raise issues and admins to track resolution status.

## 🛠️ Local Setup
1. Clone the repository: `git clone https://github.com/YugTheDevloper/societyLink.git`
2. Configure your database credentials in `src/main/resources/application.properties`.
3. Run the application via your IDE or using Maven: `./mvnw spring-boot:run`
